# Doubling / Halving

本专题包含两条独立实验线：

- `check_half_and_double.py`
  从真实 fetal 数据中挖掘 doubling / halving 候选。
- `build_synthetic_doubling_halving_dataset.py`
  从 clean dataset 合成 doubling / halving 注入数据。

默认产物目录：

- 候选挖掘：
  `artifacts/results/doubling_halving/candidate_mining/`
- 人工复核样本：
  `artifacts/results/doubling_halving/review_samples/`
- 合成数据：
  `artifacts/datasets/doubling_halving/synthetic_output/`
